### Problem Statement

Create a User class that tracks login status.

### Requirements

## Properties:

username: string

isLoggedIn: boolean (initially false)

## Methods:

login()

If user is already logged in → print "Already logged in"

Otherwise → set isLoggedIn to true

logout()

If user is already logged out → print "Already logged out"

Otherwise → set isLoggedIn to false

getStatus()

Returns "Logged in" or "Logged out"
